## Node + Express Server

1. create server.js file

2. $ npm init -y

3. npm install express

4. start => $ node server.js

### Create React Project
``` bash
// Create New Project
$ npm create vite@latest "my project name" --template react-ts

// Select
1. React
2. TypeScript

// Start
1. $ cd "my project name"
2. $ npm install
3. $ npm run build
```
